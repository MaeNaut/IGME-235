BATTLEFIELD 1

PUBLISHING INFO

  • Published by: Electronic Arts, Inc.
  • Developed by: EA Digital Illusions CE AB
  • Released: October 20th, 2016

DESCRIPTION

Battlefield 1 is the fifteenth main game in the Battlefield shooter franchise and as a main entry the follow-up to the 2013 Battlefield 4. This time it is set during World War I and it is inspired by the historic events. [...]


GAME CATEGORIES

  • Genre: Action
  • Perspective: 1st-person, Behind view
  • Gameplay: Shooter, Stealth
  • Setting: Historical events, World War 1
